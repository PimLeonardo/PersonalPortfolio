Sobre

Um pouco sobre mim, minhas experiências recentes e mais sobre minha jornada até aqui.

Minha paixão pela tecnologia começou logo cedo quando criança, interessado e curioso com tudo que era relacionado à ela. Tinha uma máquina em casa, porém não tinha acesso à internet, dessa forma, costumava frequentar casas de amigos e a LAN house, passava horas acessando à rede e pesquisando sobre diversos assuntos. Assim, mesmo depois de maior, continuei me envolvendo com a tecnologia, de modo que o desejo de me conectar à ela foi se aumentando cada vez mais.

Durante a escola sempre fui muito envolvido com grupos de pessoas que também estavam conectadas com a tecnologia, já era muito comum ficar várias horas em frente a o computador desde novo. Durante o ensino médio, surgiu uma oportunidade de cursar o curso técnico de tecnologia da informação no Instituto Federal Fluminense da minha cidade, lá pude realmente descobrir mais sobre esse mundo e especialmente sobre como eu gostava de desenvolvimento, então no momento de ingressar em um curso superior decidi seguir esse caminho.

Durante a faculdade sempre fui muito antenado e envolvido com todos os projetos e oportunidades que apareciam, participei e ajudei com diversos eventos e projetos, por exemplo, desenvolvimento de reconhecimento facial para a semana acadêmica do curso, fui gerente de marketing da empresa júnior Union Soluções e realizei alguns projetos para clientes, fiz parte eventos de hackathon com amigos do curso e também realizava trabalhos freelancer para ONGs através da plataforma FreeHelper.

Minhas habilidades

Um pouco das tecnologias que eu trabalho e estudo.

HTML

HTML (HyperText Markup Language) é o bloco de construção mais básico da Web. Ele define o significado e a estrutura do conteúdo da web.

CSS

Cascading Style Sheets (CSS) é uma linguagem de folha de estilo usada para descrever a apresentação de um documento escrito em HTML. CSS descreve como os elementos devem ser renderizados na tela, no papel, na fala ou em outras mídias.

JavaScript

JavaScript é uma linguagem de programação interpretada e estruturada, de script em alto nível com tipagem dinâmica fraca e multiparadigma. Juntamente com HTML e CSS, o JavaScript é uma das três principais tecnologias da World Wide Web.

PHP

PHP é uma linguagem de script de uso geral voltada para o desenvolvimento da web, favorecendo a conexão entre os servidores e a interface do usuário.

TypeScript

TypeScript é uma linguagem de programação de código aberto desenvolvida pela Microsoft. É um superconjunto sintático estrito de JavaScript e adiciona tipagem estática opcional à linguagem.

React

O React é uma biblioteca front-end JavaScript de código aberto com foco em criar interfaces de usuário em páginas web.

Node.js

Node.js é um software de código aberto, multiplataforma, baseado no interpretador V8 do Google e que permite a execução de códigos JavaScript fora de um navegador web.

Tailwind

Tailwind CSS é uma estrutura CSS de código aberto. A principal característica desta biblioteca é que, ao contrário de outros frameworks CSS como o Bootstrap, ela não fornece uma série de classes predefinidas para elementos como botões ou tabelas.

Git

Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software, mas pode ser usado para registrar o histórico de edições de qualquer tipo de arquivo.
